# E-commerce Furniture Website

This is a static e-commerce website for furniture, featuring various categories like bedroom, dining, hall, kitchen, mattresses, and special offers.

## Features

- Homepage with slider and navigation
- Category pages: Bedroom, Dining, Hall, Kitchen, Mattress
- Offers page with promotions
- Payment page for checkout
- Login page for user authentication
- Responsive design with CSS styling
- Interactive elements using JavaScript

## File Structure

- `index.html` - Homepage
- `bedroom.html`, `dining.html`, `hall.html`, `kitchen.html`, `mattress.html` - Category pages
- `offers.html` - Special offers page
- `payment.html` - Payment/checkout page
- `login1.html` - Login page
- Corresponding `.css` and `.js` files for each page
- `header.html` and `header.css` - Shared header component

## Setup and Running

1. Clone or download the project files.
2. Open `index.html` in a web browser to view the website.
3. No server setup required as it's a static site.

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
